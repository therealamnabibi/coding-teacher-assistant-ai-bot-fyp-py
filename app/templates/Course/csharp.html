C# Notes

Comprehensive and beginner-friendly notes on C# programming.

Introduction to C#

C# is a modern, object-oriented programming language developed by Microsoft. It is widely used for developing desktop applications, web applications, and games. C# offers robust features, making it an ideal choice for developers to build versatile and high-performance software.

Key Features of C#

  • Object-oriented and type-safe language.
  • Simple and easy-to-learn syntax.
  • Supports modern programming paradigms like LINQ and asynchronous programming.
  • Integrated with .NET framework for seamless development.
  • Cross-platform development capabilities with .NET Core.
  • Extensive library support for faster development.

Basic Syntax


// A simple C# program
using System;

class Program {
    static void Main() {
        Console.WriteLine("Hello, World!");
    }
}

Above is an example of a basic C# program. It prints "Hello, World!" to the console, demonstrating the simplicity of the C# syntax.

Data Types in C#

Type Description
int Stores whole numbers.
float Stores fractional numbers (single-precision).
double Stores fractional numbers (double-precision).
string Stores text.
bool Stores true or false values.
char Stores single characters.

Control Statements

C# provides various control statements for decision-making and looping:

  • if-else: Used for conditional execution.
  • switch: Used for multi-way branching.
  • for, while, do-while: Used for looping.
  • foreach: Iterates through elements in a collection.

These control statements help in creating efficient and flexible programs.

Object-Oriented Programming

C# follows the principles of object-oriented programming:

  • Encapsulation: Bundling data and methods into objects.
  • Inheritance: Sharing functionalities between classes.
  • Polymorphism: Using a single interface to represent different types.
  • Abstraction: Hiding implementation details and showing only functionality.

Common Use Cases of C#

C# is widely used in various industries for:

  • Developing desktop applications using Windows Forms or WPF.
  • Building web applications with ASP.NET.
  • Creating games using Unity Game Engine.
  • Working on cross-platform mobile apps with Xamarin.
  • Automating tasks and creating scripts.

Advanced Features of C#

C# offers advanced features for robust programming:

  • Delegates: Reference types for encapsulating methods.
  • Events: Handling user actions or system occurrences.
  • LINQ: Simplifies data queries with integrated syntax.
  • Asynchronous Programming: Enhances application responsiveness.
  • Reflection: Inspect and interact with object metadata.

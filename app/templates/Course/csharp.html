Comprehensive and beginner-friendly notes on C# programming.

Introduction to C#

C# is a modern, object-oriented programming language developed by Microsoft. It is widely used for developing desktop applications, web applications, and games.

Key Features of C#

  • Object-oriented and type-safe language.
  • Simple and easy-to-learn syntax.
  • Supports modern programming paradigms like LINQ and asynchronous programming.
  • Integrated with .NET framework.

Basic Syntax


// A simple C# program
using System;

class Program {
    static void Main() {
        Console.WriteLine("Hello, World!");
    }
}

Data Types in C#

Type Description
int Stores whole numbers.
float Stores fractional numbers (single-precision).
double Stores fractional numbers (double-precision).
string Stores text.
bool Stores true or false values.

Control Statements

C# provides various control statements for decision-making and looping:

  • if-else: Used for conditional execution.
  • switch: Used for multi-way branching.
  • for, while, do-while: Used for looping.

Object-Oriented Programming

C# follows the principles of object-oriented programming:

  • Encapsulation
  • Inheritance
  • Polymorphism
  • Abstraction

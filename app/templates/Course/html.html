Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. It structures the content of a webpage using elements and tags.

Key Features of HTML

  • Easy-to-learn and widely used language.
  • Platform-independent and works across all browsers.
  • Forms the backbone of every web page.
  • Supports multimedia integration (images, videos, and audio).
  • Facilitates hyperlinking between web pages.

Basic HTML Structure

<!DOCTYPE html>
<html>
    <head>
        <title>My First HTML Page</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
        <p>Welcome to HTML programming.</p>
    </body>
</html>

Hello, World!

Welcome to HTML programming.

Common HTML Tags

Tag Description
<h1> to <h6> Defines headings from largest (<h1>) to smallest (<h6>).
<p> Defines a paragraph.
<a> Defines a hyperlink.
<img> Embeds an image.
<ul> & <ol> Defines unordered and ordered lists, respectively.

Attributes in HTML

Attributes provide additional information about HTML elements:

  • id: Specifies a unique identifier for an element.
  • class: Specifies a class name for an element.
  • src: Specifies the source file for images or videos.
  • href: Specifies the URL for hyperlinks.
  • alt: Provides alternative text for images.

Forms in HTML

HTML forms are used to collect user input:

<form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    <br>
    <button type="submit">Submit</button>
</form>

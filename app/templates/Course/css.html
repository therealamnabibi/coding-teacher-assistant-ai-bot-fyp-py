Introduction to CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. It controls the layout, colors, and fonts, making web pages visually appealing.

Key Features of CSS

  • Separates content (HTML) from design.
  • Offers reusability across multiple pages.
  • Provides responsive designs for all screen sizes.
  • Supports animations and transitions.
  • Customizes elements with advanced styling techniques.

Basic CSS Syntax

/* A simple CSS example */
body {
    background-color: #f0f0f0;
    color: #333;
    font-family: Arial, sans-serif;
}

h1 {
    color: #007bff;
    text-align: center;
}

p {
    font-size: 16px;
}

Example: Changes the background color, text color, and font style of the webpage.

CSS Selectors

Selector Description
* Selects all elements.
#id Selects an element with a specific ID.
.class Selects elements with a specific class.
tag Selects all elements of a specific tag.
element1, element2 Selects all specified elements.

CSS Box Model

The CSS Box Model describes the structure of an element, consisting of:

  • Content: The actual content of the element.
  • Padding: Clears an area around the content.
  • Border: A border surrounding the padding.
  • Margin: Clears an area outside the border.

CSS Flexbox

The Flexbox Layout is a CSS layout model for distributing space and aligning items in a container:

  • justify-content: Aligns items horizontally.
  • align-items: Aligns items vertically.
  • flex-direction: Defines the main axis (row or column).
/* CSS Flexbox Example */
.container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}

.box {
    width: 100px;
    height: 100px;
    background-color: #007bff;
    margin: 10px;
}

JavaScript Notes

Comprehensive and beginner-friendly notes on JavaScript programming. JavaScript is essential for creating interactive and dynamic web pages.

Introduction to JavaScript

JavaScript is a versatile programming language used both on the client-side and server-side. It enhances the interactivity and responsiveness of websites.

Key Features of JavaScript

  • Client-side and server-side scripting capabilities.
  • Works seamlessly with HTML and CSS to create dynamic web pages.
  • Lightweight and interpreted.
  • Supports event-driven, object-oriented, and functional programming.
  • Extensive community support and rich libraries like React, Angular, and Node.js.

JavaScript Hello World

// A simple JavaScript program
console.log("Hello, World!");

Variables in JavaScript

Variables store data values. JavaScript provides three ways to declare variables: var, let, and const.

var x = 5;
let y = 10;
const z = 15;
console.log(x, y, z);

Loops in JavaScript

Loops are used to perform repetitive tasks efficiently.

for (let i = 0; i < 5; i++) {
    console.log(i);
}

Arrays in JavaScript

Arrays store multiple values in a single variable. They are versatile and support various operations like adding or removing elements.

const fruits = ["Apple", "Banana", "Cherry"];
fruits.push("Date");
console.log(fruits);

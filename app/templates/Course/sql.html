SQL Notes

Comprehensive and beginner-friendly notes on SQL programming. SQL is essential for managing and querying relational databases.

Introduction to SQL

SQL (Structured Query Language) is a standard language for accessing and manipulating databases. It is widely used for querying, updating, and managing relational databases.

Key Features of SQL

  • Allows retrieval, insertion, updating, and deletion of data in databases.
  • Follows ANSI standard and widely supported by database systems.
  • Supports data definition and manipulation (DDL, DML).
  • Works with relational database models using tables.
  • Provides advanced querying with joins, subqueries, and aggregate functions.

SQL Query Example

-- Basic SELECT Query
SELECT *
FROM users
WHERE age > 20;

SQL Clauses

SQL provides various clauses to filter and manipulate data:

  • SELECT: Retrieves data from one or more tables.
  • WHERE: Filters records based on a condition.
  • ORDER BY: Sorts results in ascending or descending order.
  • GROUP BY: Groups rows sharing a property for aggregate calculations.
  • HAVING: Filters groups based on aggregate conditions.

SQL Joins

Joins are used to combine data from two or more tables:

  • INNER JOIN: Returns matching records from both tables.
  • LEFT JOIN: Returns all records from the left table and matching records from the right.
  • RIGHT JOIN: Returns all records from the right table and matching records from the left.
  • FULL JOIN: Returns all matching and non-matching records from both tables.

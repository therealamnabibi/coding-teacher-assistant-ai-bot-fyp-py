PHP Notes

PHP (Hypertext Preprocessor) is a widely-used open-source server-side scripting language designed for web development. It's a versatile language that powers dynamic and interactive websites and applications.

Why Learn PHP?

  • PHP is easy to learn and suitable for beginners in server-side scripting.
  • It integrates seamlessly with databases like MySQL for dynamic content generation.
  • It powers over 75% of websites globally, including popular platforms like WordPress.
  • PHP offers extensive frameworks like Laravel and CodeIgniter for efficient development.

PHP Hello World

<?php
// A simple PHP script
echo "Hello, PHP!";
?>

Variables in PHP

PHP variables start with a $ symbol and store data of different types such as strings, integers, and arrays.

<?php
$name = "John";
$age = 30;
echo "Name: $name, Age: $age";
?>

Conditional Statements in PHP

PHP supports conditional statements like if, else, and switch for decision-making:

<?php
$num = 10;
if ($num % 2 == 0) {
    echo "The number $num is Even";
} else {
    echo "The number $num is Odd";
}
?>
